# Branding
Talk about the Backdrop brand and any changes that need to be made.

Whether it's about the logo, the name or the presense of the brand out in the wild, we discuss all things related to Backdrop branding here.

## Where to start...

1) If you want to discuss things, head over to the **Discussions** tab.

2) If you want to add or ammend a policy, create an issue in the **Issues** tab.

3) If you want to see the progress of any outstanding issues, check out the **Projects** tab.
